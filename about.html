Academic Innovation Project

About EduPredict AI

An intelligent Personal Academic Assistant designed to shift educational technology from passive monitoring to proactive, data-driven student intervention.

Project Mission

Traditional educational technology has long been confined to a reactive role — logging grades, recording attendance, and presenting historical data to students after the damage has already been done. EduPredict AI was developed to fundamentally challenge this paradigm.

The system's core mission is to enable proactive intervention: to analyze a student's current academic and lifestyle metrics in real time and generate an optimized, actionable plan before performance degrades. By leveraging machine learning, the assistant predicts academic outcomes and immediately scaffolds a personalized 7-day study schedule that prioritizes cognitive health — enforcing adequate sleep, distributing deep-work sessions using the Pomodoro technique, and ensuring time for rest and recovery.

Beyond prediction, EduPredict AI aims to prevent burnout — a leading cause of academic failure — by intelligently balancing study load against biological needs, ensuring students remain consistent, focused, and motivated throughout their academic journey.

Technical Architecture

EduPredict AI is built on a lean, portable, and fully open-source technology stack, designed to run as a standalone application with zero external service dependencies.

Backend & Routing

Built with Python 3 and the Flask microframework. Application logic is organized via Flask Blueprints, and authentication is managed through server-side sessions with Werkzeug password hashing.

Machine Learning Engine

Powered by Scikit-Learn, utilizing a tuned Random Forest Regressor optimized via GridSearchCV with 5-fold cross-validation. The model analyzes 9 distinct student metrics to predict a final performance score. Input features are normalized using a StandardScaler pipeline.

Database Layer

Utilizes SQLite — a fully portable, file-based relational database — accessed through the Flask-SQLAlchemy ORM. This design requires no database server, making the system trivially deployable and migratable across machines.

Frontend & UI

Server-side rendered Jinja2 HTML templates styled with Tailwind CSS via CDN, requiring no build step. Dynamic data visualizations are rendered client-side using Chart.js, and Markdown notes are processed server-side with the Python Markdown library.

Core System Pillars

📊

Predictive Analytics

The ML engine accepts nine student input parameters — including study hours, attendance, sleep quality, assignment completion rate, class participation, and parental support — and returns a continuous predicted final score alongside a categorical performance label (Low, Average, Good, Excellent). Every prediction is persisted to the database and visualized as a longitudinal trend line on the dashboard, enabling students to track their improvement over time.

🗓️

Algorithmic Timetable Generation

When a student's predicted score falls below 75%, or when key input metrics indicate risk (e.g., low attendance or insufficient study hours), the system automatically generates a personalized 7-day timetable. The scheduling algorithm operates using a strict datetime cursor — advancing block-by-block — to produce a chronologically ordered 24-hour day that mandates 8 consecutive hours of sleep, allocates time for meals, and distributes targeted Deep Study sessions using the Pomodoro technique (50 minutes focused work / 10 minutes rest). High-performers (score > 85%) are also allocated leisure and recovery time to prevent burnout.

📒

Unified Study Vault

The Study Vault is a context-aware, Markdown-based note-taking system embedded directly in the dashboard. Notes can be written in standard Markdown syntax and optionally linked to a specific timetable block via a Foreign Key relationship in the database. This allows students to attach reading summaries, formulas, or task notes directly to the corresponding scheduled activity. Saved notes are rendered as formatted HTML in real time, providing a clean and readable knowledge repository.
